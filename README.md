sublimetext-plugins
===================

Plugins for Sublime Text 2

# Installation

Copy folder to your packages folder. Then add the following to `[SUBLIME_DIR]/Packages/User/Preferences.sublime-settings`:

    ...
    ...
    "tab_size": 2,
    "translate_tabs_to_spaces": true,
    "expand_tabs_on_save": true
    ...
    ...
